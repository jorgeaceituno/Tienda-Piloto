# UI/UX & Design System

1. **Consistencia (Variables CSS):** Usa `:root` para definir los colores primarios, secundarios, bordes, espaciados y tipografías.
2. **Feedback Visual:** Implementa *spinners* y estados de carga ('loading'). Muestra *toasts* o notificaciones temporales en éxitos (ej: Producto añadido).
3. **Manejo de Espacios:** Usa un sistema de espaciado consistente (`.gap-md`, `.m-4`) basado en una escala base lógica (4px, 8px, 16px).
4. **Responsive Default:** El diseño debe enfocarse primeramente en funcionar en móviles antes de escalar a monitores e incorporar *Hover States* claros.
